== Description ==

SB Post Widget is a plugin that allows to show custom post on sidebar. You can select the widget options to display custom information of post.

**Required Plugin**

* [SB Core](https://wordpress.org/plugins/sb-core/)

**Features**

* Get post by many types: random posts, recent posts, post by category,...
* Allow to display post date, post author, post excerpt,...
* Custom size for post thumbnail.

**Translations**

* English
* Vietnamese

**Recommended WordPress Plugins**

* [SB TBFA](https://wordpress.org/plugins/sb-tbfa/)
* [SB Clean](https://wordpress.org/plugins/sb-clean/)
* [SB Comment](https://wordpress.org/plugins/sb-comment/)
* [SB Paginate](https://wordpress.org/plugins/sb-paginate/)

== Installation ==

Upload the SB Clean plugin to your blog, activate it.

1, 2, 3: You're done!

Go to the dashboard and access widgets page, drag and drop SB Post into the widget area where you want the list posts to be displayed.
